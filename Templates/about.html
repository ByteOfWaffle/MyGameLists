About MyGameLists

MyGameLists is a school project created in 2025 that aims to provide gamers with a platform to track and share their gaming experiences.

What We Offer

  • Track games you've played, are playing, or want to play
  • Rate and review games
  • See what others think about different games
  • Discover new games through community recommendations

Project Information

This website was created as a school project using:

  • Python Flask for backend
  • HTML/CSS for frontend
  • MariaDB for database
  • RAWG API for game data
